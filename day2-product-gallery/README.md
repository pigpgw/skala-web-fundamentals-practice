# 반응형 상품 카드 갤러리

## 📖 프로젝트 소개

상품 카드 갤러리를 CSS Grid로 배치하고 반응형, 다크모드, hover, 등장 애니메이션을 구현하는 Day 2 종합 실습 프로젝트입니다.

* 영역: CSS 심화

---

## ✅ 필수 요구사항

* `:root`에 5개 이상의 색상 변수를 정의하고 전체 스타일에서 `var()` 사용
* 상품 카드 6개 이상을 `display: grid`와 `repeat(3, 1fr)`로 3열 배치
* 상단바와 카드 정렬에 Flexbox의 `space-between`, `margin-left: auto` 사용

## ✅ 세부 요구사항

* `@media`로 900px 이하에서 2열로 변경
* `@media`로 600px 이하에서 1열로 변경하고 메뉴 숨김
* 카드 hover 시 `transform: translateY()`와 강화된 그림자를 `transition`과 함께 적용
* 히어로 문구에 `@keyframes` 등장 애니메이션 적용
* 다크모드 버튼으로 테마 전환

---

## 🧪 기본 동작 테스트

* 화면 폭을 1200px → 800px → 500px로 변경하면 카드가 3열 → 2열 → 1열로 바뀌는지 확인
* 500px에서 상단 메뉴가 숨겨지는지 확인
* 🌙 다크 버튼을 클릭하면 배경, 카드, 글자색이 한 번에 어두워지고 버튼이 ☀️로 변경되는지 확인
* 카드에 마우스를 올리면 카드가 위로 떠오르고 그림자가 진해지는지 확인

## 🧪 검증·결과 테스트

* 새로고침하면 히어로 제목과 부제가 아래에서 순차적으로 등장하는지 확인
* 상단바에서 로고는 왼쪽, 메뉴와 버튼은 오른쪽으로 정렬되는지 확인
* OS에서 ‘동작 줄이기’를 켜고 새로고침하면 애니메이션이 비활성화되는지 확인
