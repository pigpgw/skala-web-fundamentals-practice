# Frontend Fundamentals

> HTML, CSS, JavaScript의 핵심 개념을 과제 기반으로 학습하는 프론트엔드 실습 저장소입니다.

## 📖 소개

이 저장소는 프론트엔드 개발의 기본기를 익히기 위해 진행한 **과제 기반 실습 프로젝트**를 정리한 저장소입니다.

각 과제는 실제 요구사항을 분석하고 구현하는 과정으로 진행되며, HTML, CSS, JavaScript의 핵심 개념을 자연스럽게 익히는 것을 목표로 합니다.

---

## 🎯 학습 목표

* HTML 문서 구조와 시맨틱 마크업 이해
* Form과 다양한 Input 요소 활용
* CSS 레이아웃(Flexbox, Grid)
* 반응형 웹 구현
* CSS 애니메이션 및 사용자 인터랙션
* JavaScript를 이용한 DOM 조작
* 이벤트 처리와 상태 관리
* ES Module 활용
* LocalStorage를 이용한 데이터 영속화
* Fetch API를 이용한 비동기 처리

## 🎓 학습 방식

각 프로젝트는 아래 과정을 기준으로 진행합니다.

3. 기능 구현
4. 요구사항 체크리스트 검증
