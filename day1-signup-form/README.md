# 회원가입 & 결과 페이지

## 📖 프로젝트 소개

회원가입 폼을 만들고, 제출 시 입력값을 표로 보여주는 `result.html` 결과 페이지로 이동하는 Day 1 종합 실습 프로젝트입니다.

* 영역: HTML Form + CSS 기초
* 난이도: 중상

---

## ✅ 필수 요구사항

* 이름, 아이디, 비밀번호, 이메일, 약관 동의에 `required` 적용
* 아이디는 `pattern`으로 영문과 숫자 4~12자만 허용
* 비밀번호에 `minlength="8"` 적용
* 전화번호에 `010-0000-0000` 형식의 `pattern` 적용

## ✅ 세부 요구사항

* 성별은 `radio`로 단일 선택
* 관심 분야는 `checkbox`로 복수 선택
* 지역은 `select` 사용
* 상단 헤더를 `position: fixed`와 `z-index`로 고정
* 제출 시 `result.html`로 이동하여 입력값을 `table`로 표시
* 관심 분야 복수 값을 결과에 모두 표시

---

## 🧪 기본 동작 테스트

* 아무것도 입력하지 않고 제출하면 브라우저 경고가 표시되고 제출이 차단되는지 확인
* 아이디 2자는 패턴 위반 경고가 표시되고 4자 이상은 통과하는지 확인
* 비밀번호 3자 입력 시 8자 이상 안내가 표시되는지 확인

## 🧪 검증·결과 테스트

* 관심 분야에서 “프론트엔드”와 “디자인”을 선택하면 결과에 함께 표시되는지 확인
* 필수값을 정상 입력하면 `result.html`로 이동하고 값이 표에 정확히 표시되는지 확인
* 결과 페이지를 스크롤해도 상단 헤더가 계속 고정되는지 확인
