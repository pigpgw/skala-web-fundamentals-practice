# 여행지 소개 페이지

## 📖 프로젝트 소개

시맨틱 구조로 여행지 소개 페이지를 구현하는 Day 1 종합 실습 프로젝트입니다.

* 영역: HTML 심화(Semantic·Media) + CSS 기초
* 난이도: 중상

---

## ✅ 필수 요구사항

* 전체 구조를 `header`, `nav`, `main`, `section`, `article`, `aside`, `footer` 시맨틱 요소로 구성
* 내비게이션을 `position: fixed`로 고정하고 링크를 각 섹션의 `id`로 연결

## ✅ 세부 요구사항

* 명소 갤러리를 `figure`와 `figcaption` 3개 이상으로 구성
* 먹거리 섹션을 본문 `article`과 팁 `aside`의 2단으로 배치

## ✅ 추가 요구사항

* `video` 요소에 `controls`와 `poster` 적용
* 히어로 배경에 gradient 오버레이와 이미지 적용
* 명소 카드에 마우스를 올리면 이미지가 `scale`로 확대되도록 구현

---

## 🧪 기본 테스트

* 내비게이션의 “먹거리”를 클릭하면 해당 섹션으로 부드럽게 스크롤되는지 확인
* 아래로 스크롤해도 상단 내비게이션이 계속 고정되는지 확인

## 🧪 검증 테스트

* 명소 카드에 마우스를 올리면 이미지가 살짝 확대되는지 확인
* 푸터의 “↑ 맨 위로”를 클릭하면 페이지 최상단으로 이동하는지 확인

## 🧪 결과 확인 테스트

* 이미지 로딩을 차단하거나 개발자 도구에서 모든 `img`에 의미 있는 `alt`가 있는지 확인
* 영상 섹션의 재생 버튼으로 재생과 정지가 동작하는지 확인(인터넷 필요)

---
