# 인터랙티브 할 일 관리 앱

## 📖 프로젝트 소개

할 일을 추가·완료·삭제·필터링하고 데이터를 저장하며 오늘의 한마디를 불러오는 Day 2 종합 실습 프로젝트입니다.

* 영역: JavaScript(기초·심화)
* 난이도: 중상

---

## ✅ 필수 요구사항

* 추가 버튼 또는 Enter 키로 할 일을 추가하고 빈 값 방지
* 체크박스로 완료 상태를 토글하여 취소선을 표시
* `✕` 버튼으로 할 일 삭제

## ✅ 세부 요구사항

* 전체 / 진행 중 / 완료 필터 구현
* 하단에 전체 개수와 완료 개수 요약 표시
* 부모 `ul`에 리스너를 한 번 등록하는 이벤트 위임으로 동적 항목 처리

## ✅ 추가 요구사항

* `storage.js`는 데이터, `app.js`는 화면을 담당하도록 모듈 분리
* `type="module"` 사용
* `localStorage`로 데이터를 저장하고 복원
* 오늘의 한마디를 `fetch`로 불러오고 실패하면 기본 문구 표시

---

## 🧪 기본 테스트

* Live Server 또는 `python -m http.server`로 실행하여 모듈 오류가 없는지 확인
* 입력 후 Enter를 누르면 목록에 추가되고 입력창이 비워지는지 확인

## 🧪 검증 테스트

* 체크박스를 클릭하면 취소선이 표시되고 하단 완료 개수가 1 증가하는지 확인
* ‘완료’ 필터를 클릭하면 완료 항목만 표시되는지 확인

## 🧪 결과 확인 테스트

* `✕`로 항목을 삭제할 수 있는지 확인
* 항목 추가 후 새로고침해도 `localStorage`를 통해 유지되는지 확인
* 상단에 오늘의 한마디가 표시되는지 확인
* 인터넷 차단 시 기본 문구로 대체되는지 확인
